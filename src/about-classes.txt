Description of classes

User:
- сохраняет данные пользователя в БД;
- хранит роль пользователя;
- отображает профиль пользователя с ролью  на странице Профиля; 
- пользователь с ролью Customer:
    - создает new Task;
    - отменяет new Task [если он автор];
    - принимает Response от пользователя [переводит Task в статус Work] 
      и устанавливает ему роль Performer;
    - завершает Task [статус Done] при ее выполнении;
    - оставляет Response с Feedback на пользователя Performer.
- пользователь с ролью Performer:
    - отображает профиль пользователя на странице Профиля;
    - создает Response на new Task;
    - отказывается от выполнения Task со статусом Work.

Task:
- сохраняет данные о Задании в БД;
- хранит ID Customer и ID Performer;
- хранит текущий статус Задания;
- устанавливает правила перехода между статусами Задания
  в зависимости от роли User; 
- выводит доступные действия с Заданием для каждого ее статуса.

Response:
- сохраняет Отклик в БД;
- отображает Отклик на странице.

Feedback:
- сохраняет Отзыв в БД;
- отображает Отзыв на странице.

Rating:
- выполняет расчет рейтинга пользователя.

Files:
- сохраняет файлы Задания в публичной папке;
- сохраняет файл Аватара пользователя в публичной папке;
- сохраняет ссылку на файлы (Задания или Аватара) в БД.
